## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/mhdverel/portfolio-verel.git

# Navigate to project directory
cd portfolio-verel

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```
